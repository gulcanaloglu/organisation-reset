# 4-week-organisation

This repository contains a static landing page for **The Organisation Reset**.

## Files included
- `index.html` — main page entrypoint for GitHub Pages
- `organisation-reset.html` — original page file
- `hero.MP4` — hero video asset
- `hero2.mov` — fallback hero video asset
- `about-me.PNG` — about section image
- `.gitignore` — ignores macOS `.DS_Store`

## Publish instructions
1. Create a GitHub repository (public if you want it visible to everyone).
2. Add the remote to this local repo:

```bash
git remote add origin https://github.com/<your-username>/<repo-name>.git
```

3. Push to GitHub:

```bash
git push -u origin master
```

4. Enable GitHub Pages in the repository settings, using the root branch.

Once Pages is active, the site will be available at:

`https://<your-username>.github.io/<repo-name>/`

## Notes
- If you want the page to be a standalone homepage, `index.html` is already set up.
